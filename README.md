# adventofcode-rust

advent of code with rust

# Rust commands

Rust: version  
`rustc --version`

Update  
`rustup update`

Compile and run created .exe, example  
`rustc main.rs`  
`main.exe`

Cargo: version  
`cargo --version`

Cargo: new project  
`cargo new hello_cargo`

Cargo: build  
`cargo build`  
`cargo build --release`

Cargo: run (builds and runs the program)  
`cargo run`

Cargo: check (validate that the code is correct)  
`cargo check`
